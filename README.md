# NextJsTestApp

My personal test of [NextJs](https://nextjs.org)


## Getting Started

### Installation
This app will install all required dependencies automatically. Just start the commands below in the root folder where you stored the package.
```
$ npm install
```

### Run Server
```
$ npm run dev
```
